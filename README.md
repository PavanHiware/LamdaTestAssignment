# LambdaTest Assignment - Selenium Automation

## 📌 Project Overview
This project demonstrates Selenium automation testing using Java with TestNG framework executed on LambdaTest cloud platform.

The goal of this assignment is to validate different UI functionalities available on LambdaTest Selenium Playground using automated test scripts.

---

## 🛠️ Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven
- LambdaTest Cloud Grid

## 🚀 Test Scenarios

### ✅ Test Scenario 1
- Open LambdaTest Selenium Playground
- Click on **Simple Form Demo**
- Validate URL contains `simple-form-demo`
- Enter message in input field
- Click **Get Checked Value**
- Validate displayed message matches input

---

### ⚠️ Test Scenario 2
- Open Selenium Playground
- Navigate to **Drag & Drop Sliders**
- Move slider from default value (15) to 95
- Validate slider value

**Observation:**  
- Issue with slider locator / handling (defect identified)

---

### ✅ Test Scenario 3
- Open Selenium Playground
- Navigate to **Input Form Submit**
- Click submit without filling form → validate error
- Fill all required fields
- Submit form
- Validate success message
